Welcome to Creator AI!

This application allows you to create AI-powered video content offline, giving you complete control over your creative intellectual property.

Getting Started:
1. Launch Creator AI from your Start Menu or Desktop
2. Start by training a custom model or importing a pre-trained model
3. Use the Generate tab to create videos from text prompts
4. Save your projects and generated content

Features:
- Offline AI video generation
- Custom model training
- Multiple model types (text-to-video, image-to-video, style transfer)
- Project management
- Batch generation capabilities

System Requirements:
- Windows 10 or later
- 8GB RAM (16GB recommended)
- 2GB free disk space (more for models and output)
- DirectX 11 compatible graphics card (recommended)
